## ğŸ“¦ Installation

1. **Clone the repository**


2. **Install dependencies**
   ```bash
   # For backend
   cd backend
   npm install

   # For frontend
   cd ../frontend

   ```

   - Create a MySQL database.
   - Update database credentials in `backend/config/db.js` (or `.env` file).

   ```bash
   # Start backend
   cd backend
   npm start

   # Start frontend
   cd ../frontend
   npm start
   ```
